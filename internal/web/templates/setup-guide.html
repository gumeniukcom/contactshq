Connect Your Devices

ContactsHQ includes a built-in CardDAV server. Connect your phone, tablet, or desktop app to sync contacts automatically.

Prerequisites

HTTPS is required for mobile CardDAV clients.

Use a reverse proxy like Caddy (auto-HTTPS) or nginx with an SSL certificate. See reverse proxy examples.

Caddy (recommended — automatic HTTPS):

your-domain.com {
    reverse_proxy localhost:8080
}

iPhone / iPad

Tip: If you're logged in, go to Settings → Connect Devices in the app to download a configuration profile for one-tap setup.

  1. Open Settings → Apps → Contacts
  2. Tap Contacts Accounts → Add Account
  3. Select Other → Add CardDAV Account
  4. Fill in the fields:
    • Server: your ContactsHQ domain (e.g. contacts.example.com)
    • User Name: your email address
    • Password: your app password (create one in Settings → App Passwords)
  5. Tap Next — contacts will begin syncing

macOS

  1. Open System Settings → Internet Accounts
  2. Click Add Other Account → CardDAV account
  3. Select Manual account type
  4. Fill in:
    • Server: your ContactsHQ domain
    • User Name: your email address
    • Password: your app password
  5. Click Sign In

Thunderbird

  1. Install the CardBook add-on from Thunderbird Add-ons
  2. Open CardBook → Address Book → New Address Book
  3. Select Remote → CardDAV
  4. Enter URL: https://your-domain.com/dav/you@example.com/contacts/
  5. Enter your email and app password
  6. Click Validate, then Next

Troubleshooting

Can't connect

  • Make sure your server is accessible via HTTPS
  • Check the server URL format — it should be just the domain, not the full path
  • Verify your reverse proxy is forwarding requests correctly

Authentication failed

  • Use an app password instead of your main account password
  • Double-check that your email address is entered correctly
  • Make sure the app password hasn't been deleted

Contacts not syncing

  • Check that the ContactsHQ address book is selected/enabled in your device settings
  • Try removing and re-adding the account
  • Pull down to refresh in the Contacts app
